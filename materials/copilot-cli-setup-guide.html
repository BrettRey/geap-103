Set Up GitHub Copilot CLI

GEAP 103: Basic Computer Skills — Your AI assistant in the terminal

⏱ About 15 minutes
1

Create a GitHub account

GitHub is where your project files and version history will live. It's free.

github.com/signup

Create your account

Create account
Tip: Use your Humber email. You'll need it for the free Student Developer Pack in the next step.
2

Get the Student Developer Pack (free)

This gives you GitHub Copilot for free — including the CLI.

education.github.com/pack

GitHub Student Developer Pack

The best developer tools, free for students.

Get your pack

You'll need to verify you're a student (Humber email or student ID photo).

Note: Verification can take a few minutes to a few days. You can continue to Step 3 while you wait — you just can't use Copilot until it's approved. If it takes too long, use Copilot Chat or ChatGPT in the meantime.
3

Download and install Copilot CLI

Choose your computer type:

Windows
Mac
Chromebook
Option A: Download the installer
Go to github.com/github/copilot-cli/releases
Download the Windows .exe file. Run it. Done.
Option B: If you have WinGet
Type this:
winget install GitHub.CopilotCLI
Option A: If you have Homebrew
Type this:
brew install copilot-cli
Option B: Download the installer
Go to github.com/github/copilot-cli/releases
Download the macOS file. Unpack and move to Applications.
Copilot CLI does not run easily on Chromebooks.
Use one of these instead:
copilot.microsoft.com — sign in with your Humber account
chatgpt.com — free account
claude.ai — free account
4

Log in

Open your terminal (on Windows: search for "Terminal" or "Command Prompt"; on Mac: search for "Terminal") and type:

Type this:
copilot
Welcome to GitHub Copilot CLI!
To get started, you need to log in to GitHub.
Press Enter to open github.com in your browser...

It opens your browser. Log in to GitHub. Click "Authorize." Come back to the terminal.

✓ You're connected. The terminal will say something like "Authentication successful." You only do this once.
5

Go to your project folder

Navigate to your geap-103 folder. This is the folder you set up in Week 1.

Type this:
cd OneDrive/geap-103
On Mac, your path might be different. Ask Copilot: "How do I find my OneDrive folder?"
Why this matters: When you run Copilot CLI inside your project folder, it can see your files. It knows what you're working on. That's what makes it more useful than a chatbox.
6

Talk to it

Type what you want in English. That's it.

Type this:
copilot "I want to make a weekly study schedule. I am learning English at an intermediate level. Use simple words."
I'll create a study schedule for you. Let me make a file called
study-schedule.md in your current folder...
Here's what I've created:
📄 study-schedule.md
- Monday: Reading (1 hour)
- Tuesday: Writing practice (1 hour)
- Wednesday: Listening + vocabulary (1 hour)
- ...

The file is now in your folder. You can open it, edit it, or ask Copilot to change it:

Type this:
copilot "I study best in the morning and I have class on Tuesday and Thursday. Change the schedule."

Each time you tell it more about yourself, you get a more useful result.

✓ You're ready. You have an AI assistant that works inside your project folder, sees your files, and responds to English. The more you can say about what you need, the more it can do.
