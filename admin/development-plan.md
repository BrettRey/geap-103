## Key Design Principles (from earlier notes)

1. **The "basic" skill is management** – managing AI as teammate, not coding
2. **English is the control layer** – students communicate intent; AI translates to code
3. **File system literacy is foundational** – students need to understand where files live before directing agents
4. **Language-intensive** – all interaction in English; develops communication skills
