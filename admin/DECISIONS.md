# GEAP 103 Decisions Log

## 2026-03-25

- **Triad defences for all micro-defence rounds (Weeks 4, 8, 11).** Individual defences created overflow and idle students. Triads get all students through in one 30-min block. Instructor samples ~15/21 presenters per round with observation tracking to ensure equitable coverage across rounds.

- **Assessment timeline staggered (Mollick proposal).** MD3→Wk 11, DLs→Wk 11, Git Episodes→Wk 12, Oral Reflection→Wk 13, Final Artifact→Wk 14. No week exceeds ~38%, and the heavy week (11) pairs curation with a triad block that doesn't compete cognitively.

- **Evolving criteria 3→5→7.** Week 4 diagnostic baseline uses 3 criteria. Week 8 adds CLO 2 (troubleshooting). Week 11 adds CLOs 3–4 (AI direction/evaluation). Equal weighting across rounds (~8.3% each). First three criteria constant for longitudinal comparability.

- **Oral reflection replaces written.** One-on-one, 4 min, unmediated, with pre-tagging (5 min whole-class setup). Stronger validity at B1 than written. Week 13, other students finalize artifacts. Classes above 20 may need adjusted timing.

- **Weekly speaking practice formalized.** Session review becomes a specific scenario each week, matched to the week's learning. Students say the role-setting prompt aloud in voice mode. The prompt deliberately withholds vocabulary modelling to force production.

- **Faculty prep module is a self-contained AI prompt.** Delivered as .html via GitHub Pages. Text conversation (voice mode starts a new chat, can't carry the 1,800-word setup prompt). Five challenges with teacher flips.

- **GEAP 103 is a new course, not a replacement for EAPC 106.** The design notes already said this; the README was wrong. Fixed.

- **Pandoc conversion deferred to Phase 4.** .md is the source format; .docx/.html for distribution happens once at content lock.
